# How to Convert Markdown to Word (.docx)

## EASIEST METHOD: Use Microsoft Word Directly

1. **Open Microsoft Word**
2. **File → Open**
3. **Navigate to your project folder:**
   - `C:\Users\kvanpoppel\shot-on-me-venue-portal\`
4. **Change file type to "All Files (*.*)"**
5. **Select one of these files:**
   - `BUSINESS_LAUNCH_STEP_BY_STEP.md`
   - `BUSINESS_LAUNCH_QUICK_REFERENCE.md`
   - `SHOT_ON_ME_COMPLETE_BUSINESS_PLAN.md`
6. **Word will automatically convert the markdown**
7. **File → Save As → Choose "Word Document (*.docx)"**
8. **Save with the same name but .docx extension**

**That's it!** Word handles the conversion automatically.

---

## ALTERNATIVE: Online Converter

1. Go to: **https://cloudconvert.com/md-to-docx**
2. Upload your `.md` file
3. Click "Convert"
4. Download the `.docx` file

---

## ALTERNATIVE: VS Code Extension

1. Install "Markdown PDF" extension in VS Code
2. Open the `.md` file
3. Right-click → "Markdown PDF: Export (docx)"
