## Primary Mission

**Shot On Me** is a social payment platform that allows users to send money to friends for use at participating venues, combined with social features similar to Facebook and Instagram.

## ðŸŽ¯ Key Objectives

### 1. **Payment System - Send Money to Friends**
- Users can send money to friends via phone number
- Money is sent with a personalized message (e.g., "Happy Birthday!", "This round's on me!")
- Recipients receive money in their wallet
- **NO APP REQUIRED** - Recipients can receive via SMS without downloading the app

### 2. **Escrow System - Money Protection**
- Money stays in **YOUR account** (the platform owner) until it's used
- Funds are held in escrow until redeemed at a venue
- **NO PENALTIES** - Users can hold funds indefinitely without fees
- Money only transfers from you to the venue when redeemed

### 3. **Redemption Code System**
- Each payment gets a unique redemption code (e.g., "ABCD1234")
- Recipients receive code via SMS text message
- Venues can redeem the code without Cash App or other payment apps
- Anyone with the code can use it at participating venues
- Recipients don't need to download the app to use the funds

### 4. **SMS Notification System**
- Recipients automatically receive SMS when money is sent
- SMS includes: amount, sender name, redemption code, instructions
- No app download required to receive or use funds
- Works with any phone number

### 5. **Wallet System**
- Users see their available balance
- Shows pending balance (money sent but not yet redeemed)
- Complete payment history
- Send money directly from wallet

### 6. **Social Feed (Facebook-Style)**
- Users can post updates with text, photos, videos
- Like and comment on posts
- Share moments from venues
- Live feed updates (real-time)
- Similar to Facebook news feed

### 7. **Location Tracking (Instagram-Style)**
- Find friends nearby using location services
- Map view showing where friends are
- Discover nearby venues
- Real-time location sharing (optional)
- Similar to Instagram's location features

### 8. **Venue Discovery**
- Find nearby venues with active promotions
- See venue details, hours, promotions
- Check which friends are at which venues
- Location-based discovery

### 9. **Friend System**
- Add friends
- See friends' activity
- Discover friends nearby
- Social connections

## ðŸ’° Business Model

- **Money Flow:**
  1. User sends $50 to friend
  2. $50 stays in YOUR account (escrow)
  3. Friend receives SMS with code
  4. Friend redeems code at venue

  - You (the platform) hold all funds until used
  - No money moves until venue redemption
  - Venues redeem codes, money goes from you to them
  - No fees or penalties for users
