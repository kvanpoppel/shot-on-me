## üéØ For Shot On Me Development Work

### **Budget Range: $800 - $1,200** (Best Value)
**Best for:** Most developers, handles everything smoothly

**CPU:**
- **AMD:** Ryzen 5 5600X or Ryzen 7 5700X (6-8 cores)
- **Intel:** Core i5-12400 or i5-13400 (6-8 cores)
- **Why:** Node.js, MongoDB, and multiple dev servers benefit from multi-core

**RAM:**
- **Minimum:** 16GB DDR4
- **Recommended:** 32GB DDR4 (if budget allows)
- **Why:** Running 4 servers + VS Code + browser + MongoDB = memory hungry

**Storage:**
- **Primary:** 512GB NVMe SSD (minimum)
- **Recommended:** 1TB NVMe SSD
- **Optional:** 2TB HDD for backups/archives
- **Why:** Fast boot, quick npm installs, fast file operations

**GPU:**
- **Integrated graphics is fine** (Intel UHD or AMD Radeon)
- **Optional:** Entry-level dedicated GPU (GTX 1650 / RX 6400) if you do design work
- **Why:** Development doesn't need powerful GPU unless doing video/graphics work

**Motherboard:**
- **AMD:** B550 or B650 chipset
- **Intel:** B660 or B760 chipset
- **Why:** Good balance of features and price

**Power Supply:**
- **550W - 650W** (80+ Bronze certified minimum)
- **Why:** Efficient, reliable, room for upgrades

**Case:**
- Mid-tower with good airflow
- **Why:** Keeps components cool during long coding sessions

**Best for:** Professional development, future-proof

**CPU:**
- **AMD:** Ryzen 7 7700X or Ryzen 9 7900X (8-12 cores)

- **Why:** Handles Docker, multiple VMs, heavy multitasking

**RAM:**
- **32GB DDR5** (or DDR4 if DDR5 is too expensive)
- **Why:** Run multiple projects, Docker containers, VMs simultaneously

**Storage:**
- **1TB NVMe Gen 4 SSD** (primary)
- **2TB SATA SSD** (secondary)
- **Why:** Fast primary drive, spacious secondary for projects

**GPU:**
- **RTX 3060 or RX 6600** (if doing design/video work)
- **Integrated is still fine** for pure development
- **Why:** Better for design tools, video editing, gaming breaks

**Motherboard:**
- **AMD:** X670 or B650 (DDR5 support)
- **Intel:** Z790 or B760 (DDR5 support)
- **Why:** Modern features, better connectivity

## ‚ö†Ô∏è What to Avoid

**Don't Buy:**
- ‚ùå **8GB RAM** - Too little for modern development
- ‚ùå **HDD-only** - Too slow, get at least 256GB SSD
- ‚ùå **Older CPUs** - Avoid anything older than 2020 (Ryzen 3000 / Intel 10th gen)
- ‚ùå **Unknown PSU brands** - Can damage components
- ‚ùå **OEM-only parts** - Hard to upgrade later

**Red Flags:**
- "Gaming PC" with integrated graphics only (overpriced)
- No SSD mentioned (will be slow)
- "Upgradeable" but proprietary parts
- No RAM/Storage specs listed
